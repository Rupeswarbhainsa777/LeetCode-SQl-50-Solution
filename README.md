# 🗄️ LeetCode SQL 50 Solutions

This repository contains my solutions to the **Top 50 SQL problems on LeetCode**.  
The goal of this repository is to strengthen SQL fundamentals and improve problem-solving skills using structured queries.

## 📌 About

The SQL 50 Study Plan on LeetCode is designed to cover essential SQL concepts required for technical interviews and real-world data analysis tasks.

This repository includes solutions covering:

- Basic Select Queries
- Joins
- Aggregations
- Subqueries
- Window Functions
- Advanced SQL Queries

## 🧠 Topics Covered

| Topic | Description |
|------|-------------|
| Basic Queries | SELECT, WHERE, ORDER BY |
| Filtering | DISTINCT, LIMIT |
| Joins | INNER JOIN, LEFT JOIN |
| Aggregation | COUNT(), SUM(), AVG() |
| Grouping | GROUP BY, HAVING |
| Subqueries | Nested Queries |
| Window Functions | ROW_NUMBER(), RANK() |
| Advanced SQL | Complex Interview Problems |
